# Hi, I'm Jeremy 👋

**Android kernel developer** focused on building hardened, privacy-first kernels for Qualcomm-based devices.

---

## 🔧 What I Build

- **Custom Android kernels** with root hiding, performance tuning, and advanced security
- **Automated CI/CD pipelines** using GitHub Actions for kernel compilation
- **Integration of KernelSU, SUSFS, Zeromount, NoMount, and KPM** into production-ready builds

---

## 📌 Featured Project

### [android-kernel-build](https://github.com/JWEB0689/android-kernel-build)
> Universal GitHub Actions kernel builder for the Motorola Edge+ (rtwo) — Android 13 / 5.15 KMI

**What it does:**
- Builds flashable AnyKernel3 ZIPs from any kernel source
- Supports 5 KernelSU forks (SukiSU, Tiann, KernelSU-Next, ReSukiSU, WKSU)
- Auto-applies SUSFS root hiding and Zeromount VFS-layer patches
- Bundles SUSFS module + KSU Manager APK into the ZIP
- Interactive volume-key prompts (VOL+ / VOL-) during TWRP flash
