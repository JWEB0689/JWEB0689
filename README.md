### Featured: android-kernel-build

Universal GitHub Actions kernel builder for the Motorola Edge+ (rtwo) — Android 13 / 5.15 KMI

Link: https://github.com/JWEB0689/android-kernel-build

What it does:
- Builds flashable AnyKernel3 ZIPs from any kernel source
- 5 KernelSU forks: SukiSU, Tiann, KernelSU-Next, ReSukiSU, WKSU
- SUSFS root-hiding patches + Zeromount VFS redirection
- Bundles addon modules and KSU Manager APK into the ZIP
- Volume-key prompts in TWRP recovery: VOL+ = Yes, VOL- = No
